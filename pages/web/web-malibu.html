Diseño y desarrollo de un sitio web con HTML, CSS, Bootstrap y Sass.

Se buscó desarrollar un sitio web responsive de la mano de los principios del mobile first. De esta forma, se logró una web en la que la visualización del contenido se adapta a los distintos soportes evitando ser una réplica exacta de la versión desktop. También se decidió optimizar la web en términos de SEO, aplicando desiciones referentes al meta contenido (description y keywords), estableciendo títulos (Tag Tittle) descriptivos para cada sección, y jerarquizando los titulares de cada contenido con las distintas opciones del tag h (h1, h2, h3...).
